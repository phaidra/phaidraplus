searchFilter Class

Defined in: components/search-filter.js:7
Module: search

The searchFilter class is used to create and manage a certain search input field. The searchFilter supports multiple types:

  • boolean
  • date
  • enum
  • objects
  • radio
  • select with option groups
  • text

Todos

  • add range selections
